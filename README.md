Mod with miscellaneous changes and additions made by GK1H.
<h1>Features</h1>
Wet sponges being added to fishing junk loot.<br>
Ender dragon drops dragon head.<br>
Blocking with mace. (Requires Sword Blocking Mechanics.)<br>
Fix for owls from Ender Zoology being too loud by setting volume to what it is in later versions.
